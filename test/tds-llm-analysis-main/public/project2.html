How to play:

  • Start by POSTing JSON to /submit with { email, secret, url, answer }, using url = /project2.
  • After each submission, the response tells you if it is correct and, if allowed, the next URL. Open that URL in the browser to read the task-specific instructions.
  • Each task page clearly states the required answer format to unlock the next URL. Difficulty 1–2 always reveal the next link; difficulty 3–5 reveal it only when correct.
  • Some steps are personalized to your email (noted on the step page). Steps marked “Not personalized” accept the same answer for everyone.
